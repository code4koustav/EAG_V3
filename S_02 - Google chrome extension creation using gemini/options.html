Gemini API Key

Your API key is stored locally on this device using chrome.storage.local. It is never sent anywhere except directly to the Google Gemini API.

Get your free key at ai.google.dev → "Get API key"

Model options refresh based on the Gemini API key you enter.

How to get an API Key

  1. Go to ai.google.dev
  2. Sign in with your Google account
  3. Click "Get API key" → "Create API key"
  4. Copy the key and paste it above
  5. Click Save Settings

🔒 Your key is saved using chrome.storage.local — it stays on your device only.
